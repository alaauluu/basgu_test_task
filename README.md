#Тестовое задание
Детектор объектов для военной предметной области. Обучен на базе **Ultralytics YOLO11s**.  
Классы: `armor`, `person`, `military_vehicle`, `bike`.


## 📌 Описание задачи

Детекция четырёх классов в аэрофотоснимках и снимках военной техники:

| Класс        | Описание                              |
|--------------|-----------------------------------    |
| `soldier`     | Военнослужащий                        |
| `car`        | Лёгкий военный автомобиль / Грузовики |
| `motorcycle` | Мотоцикл / Велосипед                  |
| `armor`      | Бронетехника (БТР, танк и т.п.)       |

**Выбор YOLO11s:** модель обеспечивает баланс между точностью и скоростью инференса, учитывая, что целевое edge‑устройство NVIDIA Jetson Orin Nano 8GB / NVIDIA Jetson Orin NX 8GB. В сравнении с YOLOv8s YOLO11s показывает лучший mAP при меньшем числе параметров. Вариант `s` выбран как оптимальный для ограниченного датасета (~4 400 изображений), возможностей целевого устройства.

## ❌ Анализ ошибок

**Основные паттерны ошибок:**
- Самая большая и основная проблема - модель не обучилась до конца, даже на логах четко видно, что с каждой эпохой показатели растут в положительную сторону, это произошло по причине того, что сессия в Google Colab ограничена, поэтому я подобрал то количество эпох, которые точно успеют выполнится 
- Модель путает объекты (например танк с грузовиком, человека с танком и т.д)
- Даже в достаточно простых эпизодах бывает, что модель либо не видит обьект, либо слишком маленькая уверенность
- Есть достаточно большая разность между классами по количеству аннотаций, для класса soldier их заметно меньше, чем для остальных классов, что видно в файле labels.jpg
- Еще одна заранее известная проблема, с которой я стокнулся, в классе military_vehicle преобладают изображения с грузовиками (камаз, урал и т.д.), при это в классе также встречаются изображения с легковыми автомобилями 


**Что можно улучшить:**
- дообучение модели 
- увеличение изображений с легковыми автомобилями в классе military_vehicle
- для всех классов, кроме armor нашлось очень мало изображений с дальних дистанций, особенно меня интересовали изображения, снятые с помощью беспилотного летательного аппарата, поэтому нужно найти подобные для класса bike, military_vehicle, soldier
